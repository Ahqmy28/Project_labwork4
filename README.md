## 🧁 Project Overview

This project integrates **hardware and software** into a complete IoT system.

When the touch sensor is activated:
- 💡 The light bulb automatically turns ON  
- 🌡️ Temperature is measured  
- 💧 Humidity is measured  
- 📡 ESP32 processes the data  
- 🌐 Data is sent to a web application  
- 📊 Graphs display the values in real time  

---

## 🛠 Hardware Components

| Component | Function |
|-----------|----------|
| ESP32 | Microcontroller & data processing |
| Touch Sensor | Detects user touch input |
| Light Bulb | Output device |
| SHT21 Sensor | Measures temperature & humidity |
| Jumper Wires | Circuit connection |

---

## 🔌 Pin Configuration (Example)

| Device | ESP32 Pin |
|--------|----------|
| Touch Sensor | GPIO 02 |
| Light Bulb | GPIO 16 |
| SHT21 SDA | GPIO 21 |
| SHT21 SCL | GPIO 22 |
